# Module 01 – Platform Navigation and User Interface

## Overview

This module covers the fundamental navigation and user interface components of the ServiceNow platform. The objective is to understand how administrators and users interact with the platform, locate applications, work with lists and records, and personalize their workspace.

All labs from this module were completed using a ServiceNow training instance as part of the ServiceNow Administration Fundamentals (SNAF) course.

---

## Learning Objectives

After completing this module, the following concepts were practiced:

* Navigating the ServiceNow platform
* Understanding the Application Navigator
* Working with lists and records
* Creating and applying filters
* Personalizing list layouts
* Accessing modules and applications
* Using search to locate platform components

---

## Key Platform Components

### Application Navigator

The Application Navigator is used to locate applications and modules within the platform. It provides quick access to administrative tools, system configuration areas, and application functionality.

During the lab exercises, the navigator was used to:

* Search for modules
* Access configuration areas
* Open tables and records

---

### Lists

Lists display multiple records from a table in a tabular format.

The following actions were performed in the lab:

* Viewing records in list format
* Sorting list columns
* Adding and removing columns
* Modifying list layouts

Lists are commonly used by administrators to manage large sets of records efficiently.

---

### Filters

Filters allow users to display only the records that match specific criteria.

Lab exercises included:

* Creating filters using field conditions
* Saving frequently used filters
* Applying filters to lists
* Adjusting filter logic

Filtering is essential for quickly locating relevant data within large datasets.

---

### Records and Forms

Records represent individual entries in a table. When a record is opened, it appears as a form.

During the module labs, the following tasks were performed:

* Opening and editing records
* Understanding form sections
* Navigating form fields and related lists

Forms are the primary interface used to create and update data in the platform.
