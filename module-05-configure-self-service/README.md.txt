# Module 05: Configure Self-Service

## Overview
This module focuses on enabling end-user self-service capabilities within ServiceNow. It covers configuring Knowledge Management, building and managing the Service Catalog, and implementing workflow automation to streamline service delivery. The goal is to reduce manual effort, improve user experience, and standardize request fulfilment processes.

## Learning Objectives
- Configure and manage Knowledge Base articles
- Build and maintain Service Catalog items
- Design user-friendly self-service experiences
- Implement workflow automation for request fulfilment
- Understand how different components integrate to support self-service

## Key Concepts
- **Self-Service Portal**: Interface for users to request services and access information
- **Knowledge Management**: Creation, categorization, and maintenance of knowledge articles
- **Knowledge Base (KB)**: Repository of articles for user reference and issue resolution
- **Service Catalog**: Structured list of services available to users
- **Catalog Items**: Individual requestable services with associated variables and workflows
- **Record Producers**: Simplified forms for creating records in specific tables
- **Workflow Automation**: Automating tasks, approvals, and processes using workflows/flows
- **Approval Process**: Routing requests for authorization before execution

## Lab Activities Completed
- Created and managed knowledge articles within a Knowledge Base
- Configured article categories and access permissions
- Built Service Catalog items with variables and user input forms
- Created record producers for simplified data submission
- Designed and configured workflows for catalog item fulfilment
- Implemented approval mechanisms within workflows
- Tested end-to-end request flow from submission to completion

## Outcome
By completing this module, I developed the ability to configure a complete self-service ecosystem in ServiceNow. I can create knowledge resources, design service catalog offerings, and automate workflows to ensure efficient and standardized service delivery.
